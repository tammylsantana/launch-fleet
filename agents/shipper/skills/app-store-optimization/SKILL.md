# App Store Optimization Skill

Optimize App Store listings for maximum visibility and conversion.

## Keyword Strategy
1. Use all 100 characters — every unused character is wasted opportunity
2. Do not repeat words already in the app name or subtitle
3. Use commas to separate keywords (no spaces after commas)
4. Prioritize long-tail keywords with less competition
5. Include competitor brand misspellings if ethical and allowed

## Description Best Practices
1. First sentence is critical — it shows before "Read More"
2. Use short paragraphs (2-3 sentences max)
3. Lead with benefits, not features
4. Include social proof if available (user count, ratings, awards)
5. End with a clear call to action
6. Use line breaks and bullets for scannability

## Screenshot Strategy
1. First 3 screenshots determine conversion — make them count
2. Lead with the hero screenshot (the "wow" moment)
3. Show the core value proposition in screenshot 1
4. Show key differentiator in screenshot 2
5. Show social proof or credibility in screenshot 3
6. Remaining screenshots show features
7. Always include captions — they increase conversion 15-25%

## Common Rejection Reasons to Avoid
1. Misleading screenshots (must reflect actual app functionality)
2. Placeholder content in screenshots
3. Missing privacy policy
4. Incomplete data collection disclosure
5. Crash on launch (ensure app is stable)
6. Broken links in metadata
