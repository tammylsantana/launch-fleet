# Brave Search Skill

Use Brave Search to find real-time market data, competitor information, pricing, and trends before producing any research report.

## When to Use
- Before writing any market analysis or competitor summary
- When you need current App Store data, pricing, or ratings
- When researching trending topics or keywords
- When verifying claims about market size or growth

## How It Works
You will receive search results automatically injected into your prompt context under the heading "WEB RESEARCH RESULTS (live data)". Use these results as your primary data source.

## Rules
- Always cite the URL source for data you reference from search results
- Cross-reference multiple sources when possible
- If search results contradict each other, note the discrepancy
- Never present search result snippets as your own original analysis — synthesize and add value
